1) Compile the project in the terminal using:

gcc main.c file_utils.c -o duplicateFinder


2) Run the program in the terminal using:

./duplicateFinder



3) Enter the folder path when prompted

Example:

C:\Users\YourName\Downloads


FEATURES

- Recursively scans folders and subfolders
- Detects duplicate file contents
- Detects copied filenames such as:
  file (1).txt
  file (2).txt

- Groups duplicate results together
- Filters unnecessary system/cache files
- No external libraries required


SUPPORTED FILE TYPES

- .pdf
- .docx
- .txt
- .png
- .jpg
- .jpeg
- .zip
- .mp4
- .mp3
- .exe


NOTES

- Very small files are ignored to reduce noisy results
- Duplicate groups are limited for cleaner output
- The tool compares file sizes and file contents
- Large folders may take longer to scan
